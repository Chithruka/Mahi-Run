# Mahi-Run 🍄🇱🇰

**Mahi-Run** is a satirical, fan-made HTML web game that brings the viral comedy animation created by FTT to life! It takes the nostalgic 8-bit platforming of classic *Super Mario Bros* and gives it a hilarious Sri Lankan political twist. 

**Play it live here:** [https://chithruka.github.io/Mahi-Run/](https://chithruka.github.io/Mahi-Run/)

## 🌟 Inspiration & Credits

This playable web version was built to celebrate the original comedy animation and the brilliant local internet culture behind it. 

* **Original Concept & Animation:** Created by FTT (Unlimited Content). Watch the original viral video on [YouTube](https://m.youtube.com/watch?v=uaJ1pFVaXw0).
* **Original Artwork & Character Design:** The incredible custom pixel art and character designs were created by [Sarith Ranasinghe](https://www.behance.net/gallery/158677767/Sri-Lankan-Political-Mario-Animation) (alongside Mindula Botheju).
